## 版本说明

### 版本号格式

版本号遵循语义化版本规范：`主版本号.次版本号.修订号`

- **主版本号**：不兼容的 API 修改
- **次版本号**：向下兼容的功能性新增
- **修订号**：向下兼容的问题修正

### 变更类型

- **新增**：新功能
- **变更**：现有功能的变更
- **弃用**：即将移除的功能
- **移除**：已移除的功能
- **修复**：Bug 修复
- **安全**：安全相关的修复

## 如何更新此文件

当发布新版本时，请：

1. 在顶部添加新的版本号部分
2. 将"未发布"部分的内容移到新版本下
3. 按照变更类型组织内容
4. 更新日期（格式：YYYY-MM-DD）
